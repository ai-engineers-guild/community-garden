# Community Garden: Roadmap

Проект "Community Garden" стремительно развивается от оффлайн-утилиты для ИИ-агентов к полноценному SaaS/Bot-решению для комьюнити-менеджеров.

## 🚀 Phase 1: Интеграция в Telegram (Runtime Wrapper)
- **FastAPI + Telegram Bot**: Создание рантайм-обертки над ядром проекта в виде микросервиса на FastAPI. Переход от локального запуска агентом (через CLI) к работе на сервере.
- **Поддержка LLM API**: Интеграция с API (Gemini, OpenAI, OpenRouter) на бэкенде, чтобы анализ проходил прозрачно для пользователя, без необходимости поднимать локальных агентов.

## 💬 Phase 2: Интерактивный Бот-Консультант
- **Генерация отчетов On-Demand**: Администраторы смогут запрашивать полные или краткие отчеты (WOW, Health Radar) прямо в Telegram-боте по кнопке или команде.
- **Диалог с аналитиком (Chat with Data)**: Возможность общаться с ботом в личных сообщениях, задавая вопросы по статистике: "Кто у нас сейчас главный Сорняк?", "Какие темы вызвали отток Цветов на этой неделе?", "Что посоветуешь сделать с перегрузом чата?".

## 🎯 Phase 3: Управление Целями и Историчность
- **Кастомные правила и цели**: Пользователи смогут задавать жесткие правила (Guidelines) и бизнес-цели сообщества. ИИ будет проводить аудит строго относительно этих правил.
- **Отслеживание State & History**: Ведение исторического стейта чата. Бот будет видеть тренды метрик, развитие конфликтов и эволюцию рисков на длительной дистанции (Month-over-Month).
- **Трекинг Рекомендаций (Accountability)**: В каждый новый закрытый админский отчет бот будет добавлять раздел-ретроспективу: какие риски с прошлой недели усугубились/ушли, и какие рекомендации бота были реально выполнены админами.

## 📊 Phase 4: Публичность и Данные
- **Анонимные публичные отчеты**: Автогенерация "стерильных" отчетов без раскрытия личных данных (PPL) и конкретных конфликтов, чтобы делиться здоровьем комьюнити с самими участниками.
- **Генератор графиков (Skill)**: Новый модуль, позволяющий боту "на лету" рисовать публичные графики (пироги, радары, тренды) по запросу на базе 24+ семантических метрик.

## 🌍 Phase 5: Multi-Tenant SaaS (Масштабирование)
- **Публичный Telegram Бот**: Любой желающий сможет добавить бота в свою группу и получить мощную аналитику.
- **Bring Your Own Key (BYOK)**: Пользователь просто кидает в личку боту свой токен (Gemini/OpenAI/OpenRouter). Токен безопасно шифруется и используется только для анализа его групп.
- **Мульти-Групповое управление**: Один пользователь сможет держать бота в нескольких разных чатах/проектах. В интерфейсе бота появится меню переключения между проектами для просмотра аналитики и генерации отчетов по каждому саду отдельно.
