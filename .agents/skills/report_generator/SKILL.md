# Report Generator Skill

Этот скилл собирает все данные за ДВА периода (текущий и прошлый) и генерирует исчерпывающий, структурированный Week-over-Week (WOW) отчет.

## Workflow

1. Собери ВСЕ файлы из `.garden/silver/findings/`, `.garden/silver/risks/`, `.garden/silver/metrics/` и `.garden/interventions/` за **ДВА** периода: целевую неделю (например, `2026-W26`) и предыдущую неделю (например, `2026-W25`). Ты **обязан** прочитать данные за оба периода, чтобы сделать сравнение (Week-over-Week).
2. Сформируй **Markdown-отчет** со СТРОГО следующей структурой (не пропускай ни один пункт!):
   - **TL;DR**: Самая краткая выжимка (3-4 буллита) о главных изменениях.
   - **Health Radar**: Баллы по осям, сравнение с прошлой неделей (было/стало).
   - **Все метрики таблицей**: Полная таблица ВСЕХ количественных и семантических метрик (ОБЯЗАТЕЛЬНО ВСЕ 24+ метрики из semantic_metrics_*.yml, а не только избранные) с колонками "Текущая неделя", "Прошлая неделя" и "Динамика (+/-)".
   - **Ролевой анализ**: Кто есть кто, изменение ролей конкретных людей (по именам).
   - **Трайбовый анализ**: Группировки, лидеры, конфликты.
   - **Поведенческий анализ**: Разбор деструктивного и конструктивного поведения участников (по именам).
   - **Социальный анализ**: Вовлеченность, социальные связи, отношение к новичкам.
   - **Анализ тем**: Главные обсуждаемые темы недели (возьми из `llm_packs` или выводов).
   - **Анализ действий администрации**: Оценка работы модераторов/админов, их ошибки и успехи.
   - **Инсайты**: Глубокие выводы социолога из всего вышеперечисленного.
   - **Риски**: Описание выявленных рисков с указанием имен.
   - **Рекомендации**: Конкретный, жесткий Action Plan для комьюнити-менеджера.
3. В **каждой** секции отчета **обязательно** сравнивай текущую неделю с прошлой!
4. **ОБЯЗАТЕЛЬНАЯ ВЕРИФИКАЦИЯ:** Прежде чем финализировать отчет, открой и сравни его выводы всырую с распаршенным сырым текстом логов за период в файле `.garden/gold/llm_packs/weekly_{period}.md`. Убедись, что ни одно конкретное событие (особенно баны, ридонли, отмены эфиров, действия Сорняков и Пацифистов, а также игнор Садовника) не потерялось при суммаризации агентами. Если что-то упущено — исправь черновик.
5. Сгенерируй **HTML-версию** этого отчета (с чистым, профессиональным CSS, таблицами и заголовками) и **Markdown-версию**.
6. Сохрани сгенерированные файлы в `.garden/gold/reports/` с названиями `weekly_report_WOW_{period}.md` и `weekly_report_WOW_{period}.html`.
