# Report Generator Skill

Этот скилл берет все собранные данные из `.garden/silver/` (метрики, находки других агентов) и формирует красивый финальный отчет для комьюнити-менеджера.

## Настройки по умолчанию
- **Период по умолчанию:** 1 неделя от даты запроса пользователя (например, `last-week`).
- **Сравнение:** Неделя к неделе (week-over-week).
*Примечание:* Пользователь может изменить период в своем запросе (например, попросить отчет за месяц).

## Workflow

1. Собери все `.yml` файлы из `.garden/silver/findings/`, `.garden/silver/risks/`, и `.garden/silver/metrics/` за запрашиваемый период и за *предыдущий* аналогичный период.
2. Сформируй **Markdown-отчет**, который включает:
   - Executive Summary (изменения Health Radar).
   - Список главных героев и нарушителей (по именам).
   - Динамику рисков и трайбов (что изменилось по сравнению с прошлым периодом).
   - Действия админов (Admin Mirror).
   - Рекомендации на следующий период.
3. Сгенерируй **HTML-версию** этого отчета, готовую для вставки в email, Notion или Confluence (используй чистый HTML с базовым CSS для красивого отображения).
4. Отчет должен быть объективным, с четким указанием имен и подтверждающих данных.
5. Сохрани сгенерированные файлы в `.garden/gold/reports/`.
